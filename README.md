# 🎱 大乐透选号工具

> 大乐透旋转矩阵选号小游戏，随机生成彩票号码组合方案。

## 在线体验

- 🔗 **链接**：[note.moguit.cn](https://note.moguit.cn)
- 📦 GitHub Pages：[iridescenceZz.github.io/baize](https://iridescencezz.github.io/baize/)

## 功能

- 前区号码（1-35）多选 + 胆码锁定
- 后区号码（1-12）多选
- 旋转矩阵算法生成多注组合
- 一键复制号码方案
- 移动端适配 / PWA 友好
